# Public 目录说明

## 支付二维码图片

请将您的微信支付二维码图片保存为 `payment_qr.png` 并放置在此目录下。

### 图片要求：
- 文件名：`payment_qr.png`
- 格式：PNG（推荐）或 JPG
- 建议尺寸：至少 400x400 像素，确保二维码清晰可扫

### 放置位置：
```
public/
  └── payment_qr.png  ← 将图片放在这里
```

### 注意事项：
- 图片路径在代码中引用为 `/payment_qr.png`
- 如果图片未正确放置，界面会显示占位提示
- 确保图片清晰，二维码可被微信扫描识别
